# 🎂 Cake Run

Jogo em Unity onde o jogador entrega bolos por uma cidade, cuidando para que
cheguem intactos ao destino. Projeto pessoal com objetivo duplo: aprender
Unity/C# na prática e construir um item de portfólio real.

## Status
Em desenvolvimento — veja o progresso em [`docs/Roadmap.md`](docs/Roadmap.md).

## Estrutura do repositório

```
CakeRun/
│
├── docs/
│   ├── GDD.md          ← Bíblia do jogo (design + aprendizado por bloco)
│   ├── Roadmap.md       ← Ordem e ritmo dos blocos de desenvolvimento
│   ├── DevLog.md        ← Diário de desenvolvimento (sessão a sessão)
│   ├── Ideias.md        ← Backlog de ideias futuras (fora do MVP)
│   ├── Assets.md        ← Controle de licenças de modelos/sons/fontes
│   └── Bugs.md          ← Bugs encontrados e corrigidos
│
├── UnityProject/        ← Projeto Unity em si
│
└── README.md            ← Este arquivo
```

## Como este projeto é desenvolvido

Cada funcionalidade do jogo é um "bloco" fechado, documentado no `GDD.md`,
com:
1. Objetivo claro
2. Checklist de tarefas
3. Lista de conceitos técnicos aprendidos naquele bloco
4. Um checkpoint de conhecimento — só avança se souber responder

Esse método une aprendizado de Unity/C# com produção real de jogo, em vez de
apenas seguir tutoriais passivamente.
