# 💡 Ideias Futuras — Cake Run

Backlog de ideias que **não** entram no escopo atual (MVP), mas que valem
registrar para não esquecer e não travar o foco agora.

> Regra: qualquer ideia nova durante o desenvolvimento vai **aqui primeiro**,
> nunca direto pro código. Se depois de terminar o MVP ela ainda parecer boa,
> promove pro Roadmap.

## Mecânicas
- [ ] Bicicleta/skate como meio de transporte alternativo
- [ ] Clima dinâmico (chuva atrapalha a qualidade do bolo)
- [ ] Modo multiplayer co-op (local primeiro, depois online) — dois jogadores
      carregando o mesmo bolo juntos, física sincronizada, inspirado em
      "Carry the Glass". Ordem sugerida: 1) co-op local (mesmo PC/2 controles,
      sem rede) → 2) multiplayer online (Netcode for GameObjects, Photon
      Fusion ou Mirror). Só começar depois do MVP solo estar jogável.

## Conteúdo
- [ ] Vários tipos de bolo, cada um com regras de transporte diferentes
- [ ] Sistema de reputação com os NPCs/clientes
- [ ] Customização visual do personagem

## Técnico
- [ ] Sistema de save/load de progresso
- [ ] Suporte a gamepad completo
- [ ] Versão mobile (controles touch)

## Visual/Áudio
- [ ] Trilha sonora original (compor ou encomendar)
- [ ] Efeito visual de "bolo perfeito" ao entregar sem dano
